# TIL (Today I Learned)

soichiyoの日々の学習記録をGitHubで管理するリポジトリです。  
気が向いたときに、学んだことをメモしていきます。

## このリポジトリについて

TIL（Today I Learned：今日学んだこと）は、世界中の開発者の間で10年以上前から行われている取り組みです。

- 記録を書くことで理解が深まる
- Git/GitHubの使用を習慣化できる
- GitHubに草を生やして、取り組みを見える化できる（これが一番のモチベーションかも）
- 気軽にアウトプットすることで発信自体に慣れる
