# 2026年01月06日の学習記録

## 今日学んだこと

### MCP (Model Context Protocol) とBigQueryの統合

- Claude DesktopとBigQueryをMCPプロトコルで接続する方法を実装
- OAuth認証を使用した安全な読み取り専用アクセス設定
- MCPサーバーの設定とエラー解決のプロセス
- GitHubリポジトリ（ergut/mcp-bigquery-server）を使用した実装

### Zettelkastenメソッドの実装

- ObsidianでのZettelkasten構造の構築（Inbox、Literature、Permanent、MOCsフォルダ）
- 文献ノート、永続ノート、一時ノートの使い分け
- ノート間のリンク戦略とバックリンクの活用
- Cursor Agentを使った自動化の可能性

### TwitterからMarkdownへの変換機能開発

- cursor-ide-browserのMCPを使用したTwitterコンテンツ取得
- 短縮URL（t.co）の正式URLへの展開処理
- タイトルベースのファイル名生成ロジック
- Literature Noteとしての構造化と保存

### マーケティングデータ分析

- Google SheetsでのCPA、CPC、ROAS計算
- 粗利益率とコスト効率の分析手法
- 複数期間にわたるトレンド分析
- チームとの協働によるデータ解釈

### タスク管理の改善

- ObsidianのMaster Task ListとGoogle Calendar、Todoistの統合
- コンテキストタグ（#ctx-meeting、#ctx-adminなど）による分類
- 日次タスクの優先順位付けとレビュー

- MCPプロトコルはLLMと外部データソースを接続する強力な方法
- Zettelkastenメソッドは知識管理とコンテンツ作成の両方に有効
- Twitterコンテンツの自動取得と構造化は、Literature Note作成の効率化に貢献
